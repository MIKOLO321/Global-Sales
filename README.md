# Global Sales Executive Intelligence Dashboard

## Overview

The **Global Sales Executive Intelligence Dashboard** is an interactive Power BI solution designed to convert sales data 
into actionable business insights. It provides a clear view of revenue, sales, profit, and COGS, enabling data-driven 
decision-making.

## Key Features

* KPI cards for Revenue, Sales, Profit, and COGS
* Sales and revenue analysis by country
* Monthly and yearly sales trend analysis
* Top 10 customers by sales
* Interactive filters for Education, Occupation, and Gender

## Tools Used

* Microsoft Power BI
* Power Query
* DAX

## Business Value

The dashboard enables users to:

* Monitor overall sales performance
* Identify top-performing countries and customers
* Analyze sales and revenue trends
* Explore insights through interactive filtering

## Dataset

The dataset includes sales transactions, customer information, country, dates, revenue, profit, COGS, and demographic data.
